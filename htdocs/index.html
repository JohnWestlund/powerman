Le home page de powerman

Power to the cluster!
powerman is free Linux software that controls (remotely and in parallel) switched power distribution units, performing such operations as power on, power off, and power cycle. With its wayward brother conman, it is suitable for remote operation of Linux systems in a data center or compute cluster environment. We find the combination of conman and powerman particularly useful for remote kernel development!

powerman is community supported software, covered by the the GNU General Public License.

powerman issues are discussed on the powerman-discuss mailing list.

powerman releases including source code and packages can be downloaded from sourceforge.
